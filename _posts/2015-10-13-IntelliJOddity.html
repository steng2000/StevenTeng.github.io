IntelliJ 14.1.5 Json Oddity.

Couple of findings on IntelliJ 14.1.5 running Ubuntu 15.04
    Creating a json file foo.json in the IDE, it is listed as foo.json but in the file system it is foo!!
    This is observed on Ubuntu 15.04.
    This can wreck havoc on a variety of things.
    One example:
    Using vertx factory to start verticle using "service:foo.json" The file does not exist.
    Spend hours without getting the expected result. Discover it only when examining the file system.
